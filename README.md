# 轻聊·游戏社区 (后端)

> 2026届应届生毕业设计 | Java 后端开发

## ✨ 项目简介
一个面向游戏玩家的轻量级社区平台，集成论坛发帖、攻略浏览与全局实时聊天室功能。

## 🛠️ 技术栈
- **核心框架**：Spring Boot 3
- **ORM**：MyBatis-Plus
- **数据库**：MySQL 8.0, Redis 7
- **通讯**：WebSocket
- **部署**：Docker Compose
